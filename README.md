# ShopLocal · QP

Mobile-first responsive app shell.

How to view:
1. Open C:\project\ShopLocal\QP in your editor.
2. Open `index.html` in a browser (double-click or use "Open with Live Server" in VS Code).
3. The app is pure HTML/CSS/JS — no build step required.

Notes:
- All content areas are placeholders for customization.
- Navigation is keyboard and touch friendly, with ARIA attributes for accessibility.
- CSS is mobile-first, responsive across phone/tablet/desktop.
- Colors are defined in :root for easy theming.
